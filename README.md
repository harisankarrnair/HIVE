Project Overview

HIVE is a modern, full-stack web application designed to digitize and centralize essential services for hostel residents and administration. Built with the MERN stack, it provides students with a single platform to manage everything from maintenance requests and laundry bookings to tracking mess crowd levels and registering for sports events.

üéØ Goal

To streamline communication, reduce paperwork, and enhance the overall living experience in the integrated hostel environment.

üíª Tech Stack

HIVE is built on a modern, robust architecture tailored for real-world deployment.



**Frontend (React)**

React: For the dynamic, component-based user interface.

Tailwind CSS: For efficient, fully responsive styling.

Context API: For application-wide state management (user data, dark mode, notifications).




**Backend (Node.js/Express)**

Node.js/Express.js: API server running custom routes.

Mongoose: Object Data Modeling (ODM) layer for MongoDB.

CORS, dotenv: Middleware and environment management.

Database 

MongoDB: Primary data store (via MongoDB Atlas or local instance).
